stretchly v0.1.0 released

Posted under stretchly on Sep 28 2016

·

Good news everyone, version 0.1.0 of stretchly is now available to be downloaded from github.

stretchly is cross-platform open source app that reminds you to take breaks when working with computer.

What's new?

  • rename strechly to stretchly (grammar, yay!)
  • allows only one instance of app
  • settings for microbreak (duration, interval)
  • 5 color schemes

Unfortunately, there is some problem with Linux packages, so download version v0.1.1.
